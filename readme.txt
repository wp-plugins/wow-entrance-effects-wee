Create awesome entrance effects using shortcodes.

== Description ==

A tiny plugin that helps you create On Scroll animations. You can use it to emphasize various elements on your page such as videos, photos, text, etc.

The animation starts when the element in your shortcode appears on your visitor's screen while they scroll.

The plugin is very easy to use and it comes with a live preview generator for your convenience. Just wrap the content you want to animate between [wee][/wee].

That's all! I hope you enjoy WEE!

== Installation ==

1. Upload `wow_entrance_effects.zip` folder contents to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings -> WEE! Generator for live animation previews and shortcode generation.
